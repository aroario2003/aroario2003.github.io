What is Linux?

Linux or as it is correctly refrenced GNU/Linux is a free and open source operating system, meaning it is free as in price and free as in freedom. Although, Linux may not be very popular, I and many others think it is a great operating system overall. As a two year long linux user I have found a minimal amount of problems using it daily and in Virtual Machines. Everything else aside, there is a misconception about Linux and its name, most people just call it "Linux" whereas it is really "GNU/Linux". GNU is a part of the Linux OS which integrates some of the utilities which were developed by Richard M. Stallman and the free software foundation into Linux. Linux on the other hand is a kernel which differs widely from a whole operating system.

Free Software

Free Software is a type of software where the user has the right to look at and modify the source code however they choose. The idea of free software is often misunderstood, people usually think it is only free as in price yet it is free as in freedom as well. With free software, you gain the ability to have complete control over what your software does. As I mentioned in the last section, free software was invented by Richard M. Stallman and he is the reason why Linux exists today aside from Linus Torvalds who I will talk about later in this article.

Linus Torvalds

Linus Torvalds or the creator of the Linux kernel is a respected figure within the Linux community. Linus started his development of the Linux kernel in 1991 which he was intending to make closed source yet when Richard Stallman came out with GNU and free software Linus decided that he would make the kernel FOSS which has lead to what we know today as the GNU/Linux operating system. Linus today still maintains the kernel yet he does not write much code anymore, he usually only revises the code written by other maintainers if he has the time.

The Terminal

The terminal in Linux is seen most of the time as a scary place to be. However, I will show you that this is not the case because in reality the terminal is one of the best things t exist in Linux. For example, lets say you want to go into a specific folder/directory, all you have to do is type "cd folder_name", where folder_name is the name of the folder you want to go into. Now let's say you want a list of the contents of that folder, all you have to do is type "ls" and you will get a list of most of the files in that folder. In order to list ALL of the files in the folder you must use a flag such as in this case "-a" which means you would instead type "ls -a". These types of things make the terminal more efficient than your typical graphical file manager because you are typing what you want to do instead of pointing and clicking, which results in getting work done faster.
